# 🤠 RANDOM HASH GENERATOR

## ⚙️ CONFIGURATION

- You need the gcc and nodejs (to use npm) installed.

## 🔮 COMPILING

Clone the repo, go to repo folder and then:

```shell
npm run compile
```

## 💡 RUNNING

```shell
npm run start
```

## 🧹 CLEANING BUILD

To make this you need the project compiled before.

Run this command:

```shell
npm run clean
```
